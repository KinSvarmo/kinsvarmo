# Demo Script

Setup:

```bash
pnpm keeperhub:test
pnpm demo:local
```

In another terminal:

```bash
pnpm demo:check
```

1. Open the app landing page.
2. Click `Browse Agents`.
3. Point out that the marketplace has multiple catalog agents, with preview labels on non-executable listings.
4. Open `Alkaloid Predictor v2`.
5. Click `Run Demo Analysis`.
6. Watch planner, analyzer, critic, and reporter update.
7. Show the AXL communication log and the true route labels.
8. Show the KeeperHub execution block with five successful nodes.
9. Open the dedicated result page.
10. Show confidence, key findings, structured JSON, and provenance ID.
11. Mention that final 0G contract/storage references are integrated by the 0G workstream.
